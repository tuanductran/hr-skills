# Job description prompts

Use these prompts when writing, rewriting, or improving job descriptions and job postings.

- "Write a complete job description for a [role title] that leads with the opportunity, describes success outcomes, and separates must-have from nice-to-have qualifications."
- "Rewrite this job description to remove biased or exclusionary language and improve its appeal to candidates from underrepresented groups."
- "Separate the must-have and nice-to-have qualifications in this job description to reduce unnecessary barriers and improve application rates."
- "Draft the 30/60/90-day success expectations section for a [role title] job posting."
- "Remove degree and years-of-experience requirements from this job description that aren't genuinely necessary for role success."
- "Write a team and working environment section for a [role title] job description that gives candidates a realistic preview of the team dynamic."
- "Create a compensation and benefits section for a [role title] posting that includes salary range, bonus eligibility, equity, and flexibility."
- "Rewrite this set of responsibilities to be outcome-focused rather than task-focused — describe what the person will achieve, not just what they'll do."
- "Draft an inclusive closing statement for a job posting that encourages applications from candidates with diverse backgrounds."
- "Score this job description against modern JD best practices and identify the three highest-priority improvements to make before posting."
