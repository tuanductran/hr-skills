# Data engineering hiring prompts

Use these prompts when recruiting, assessing, and evaluating data engineering, data science, and analytics infrastructure talent.

- "Draft first-round screening questions to assess a candidate's hands-on experience with SQL optimization, data modeling, and ETL pipeline orchestration."
- "Create an interview scorecard rubric for evaluating data pipeline reliability, data warehousing frameworks, and cloud data architecture in [cloud provider]."
- "Write a technical take-home assignment brief asking a candidate to design a batch and streaming data processing pipeline."
- "Draft a comparison table of data orchestration tools such as Airflow, Prefect, and Dagster for recruiters evaluating candidate technical experience."
- "Create behavioral interview questions probing how a candidate handled data quality incidents or schema migration challenges in production."
- "Draft a job description for a Senior Data Engineer highlighting data governance, data quality standards, and real-time streaming requirements."
