# Building and improving HR prompts

These prompts help HR professionals write more effective AI prompts, improve existing prompts, and build reusable prompt libraries for common HR workflows.

## Improving an existing prompt

- "Review this HR prompt and improve it using role, task, context, and format structure: [paste prompt]."
- "Rewrite this vague HR prompt to produce more specific, consistent outputs: [paste prompt]."
- "This prompt is producing [too generic / incorrect tone / wrong format] outputs. Diagnose the issue and suggest three concrete improvements: [paste prompt]."
- "Add output format instructions to this prompt so it always returns a [bullet list / numbered list / table / structured document]: [paste prompt]."
- "Add role-setting to this HR prompt so the AI responds from the perspective of an experienced [HRBP / recruiter / compensation specialist]: [paste prompt]."
- "Rewrite this prompt to produce outputs appropriate for a [new employee / senior manager / executive] audience: [paste prompt]."

## Writing prompts for specific HR tasks

- "Write a reusable prompt template for generating [job descriptions / behavioral interview questions / performance review narratives] that my HR team can reuse with minimal customization."
- "Create a prompt that generates a complete [onboarding checklist / offboarding checklist / 90-day plan] for a [role] joining [team or department]."
- "Write a prompt that produces competency-based interview questions for [role] assessing [competency], including a scoring guide for each question."
- "Design a prompt that generates a performance improvement plan for a [role] with [performance issue], using a supportive and action-oriented tone."
- "Write a prompt that generates a clear, plain-language summary of [HR policy] for a non-HR employee audience."
- "Create a prompt that writes a [change announcement / policy update / all-hands message] for a [audience] with [key message] and [desired action]."
- "Write a prompt that generates balanced performance review narratives from a list of bullet-point achievements and development areas for a [rating level] employee."

## Structuring prompts with examples (few-shot)

- "Show me how to use few-shot prompting to produce consistent [job descriptions / interview questions / offer letters] that match our company style."
- "I will give you two examples of how we write [performance reviews / offer letters / policy summaries] at our company. Use that style to produce a third: [example 1] [example 2]."
- "Here is an example of a well-written [behavioral interview question / performance review narrative / job description] from our team. Use the same format and tone for [new role or scenario]: [paste example]."

## Building system prompts for HR AI tools

- "Write a system prompt that configures an AI assistant to act as an experienced HRBP advising on [employee relations / performance / organizational design]."
- "Design a system prompt for an HR content generator that enforces [inclusive language / our company tone / a maximum length of 300 words per output]."
- "Write a system prompt for a recruiting assistant that understands our company context: [paste company and role context]."
- "Create a system prompt for an HR chatbot that answers employee questions about [leave policy / benefits enrollment / performance review process] and always escalates sensitive issues to a human HR contact."
- "Design a persistent context block for an AI tool that handles compensation analysis, ensuring it always applies [our compensation philosophy / market data sources / pay equity considerations]."

## Chain-of-thought prompts for HR analysis

- "Write a chain-of-thought prompt that guides an AI through a structured pay equity analysis for a department, covering: identify roles, compare compensation by demographic, identify gaps, and recommend actions."
- "Design a chain-of-thought prompt for evaluating a candidate's fit for [role] based on [list of competencies], where the AI assesses each competency before providing an overall recommendation."
- "Create a step-by-step prompt for diagnosing high attrition in [department or role], covering: data review, contributing factors, root causes, and recommended interventions."
- "Write a reasoning-first prompt that helps an HR team prioritize workforce risks from a list of [engagement survey data / attrition signals / performance distribution data]."

## Prompt debugging and quality improvement

- "My prompt keeps producing outputs that are too long and include [unwanted section]. Add a constraint that removes it: [paste prompt]."
- "What prompt instruction prevents the AI from including uncertain HR legal advice without a disclaimer?"
- "How do I make an AI output always include [specific section or element] that it currently omits?"
- "What prompt additions encourage the AI to acknowledge uncertainty and avoid hallucinating HR-specific facts or policies?"
- "How do I constrain an AI output to a specific length — for example, no more than 150 words per section?"
- "My prompt produces different output every time. How do I make it more deterministic and consistent across multiple uses?"

## Prompt library and governance

- "Design a prompt library structure for the HR team organized by [recruiting / performance / L&D / communications / operations] categories."
- "Write prompt quality guidelines for HR professionals covering what makes an effective prompt versus an ineffective one."
- "Create a prompt review template that HR teams can use to evaluate, test, and approve prompts before adding them to the shared library."
- "What governance guidelines should HR prompt standards include to prevent misuse of AI tools on sensitive employee matters?"
- "Write a one-page prompt usage guide for HR professionals who have never used AI tools before, covering the basics of how to write and improve a prompt."
