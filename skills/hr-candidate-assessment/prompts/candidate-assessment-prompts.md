# Candidate assessment prompts

Use these prompts when designing skill evaluations, technical take-home challenges, work sample tests, and candidate scoring frameworks.

- "Draft a technical take-home assignment specification for a [role title] with clear evaluation rubrics and time constraints."
- "Create a work sample test framework for assessing candidate problem-solving and domain expertise in [department]."
- "Write an assessment validation guide for verifying candidate scores against job competency requirements."
- "Draft candidate instructions for a live coding or architecture design session for a Senior Engineer role."
- "Create an assessment scoring matrix for evaluating technical accuracy, communication skills, and code quality."
- "Draft a post-assessment feedback email template for candidates who completed a technical assessment."
