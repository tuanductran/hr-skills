# Conducting structured reference checks

## Context

A hiring manager needs to conduct thorough reference checks for a finalist candidate for an Engineering Manager position to confirm past leadership impact and team development skills.

## Step 1: Prepare reference questions and core competencies

Sample prompt: "Draft 5 core reference check questions focused on evaluating an Engineering Manager's conflict resolution, delivery reliability, and team retention."

Expected response: A set of open-ended behavioral reference questions probing specific examples of past project delivery, team coaching, and response to project setbacks.

## Step 2: Conduct reference call and probe for specifics

Sample prompt: "Draft follow-up probing questions to ask a reference when their initial response about a candidate's performance is generic."

Expected response: Targeted follow-up probes asking for concrete metrics, specific project examples, and areas where the candidate required constructive feedback or growth.

## Step 3: Summarize findings for the hiring decision

Sample prompt: "Write a reference check synthesis template summarizing feedback from two former managers and one peer."

Expected response: A structured summary matrix highlighting consistent strengths, confirmed technical achievements, management style insights, and recommended onboarding support focus areas.

## Workflow summary

The recruiter and hiring manager prepare competency-aligned reference questions, conduct structured reference conversations, and synthesize findings to validate the final hiring decision.
