# Reference checking prompts

Use these prompts when verifying candidate work history, conducting structured reference interviews, assessing managerial references, and evaluating candidate performance records.

- "Draft a structured reference check questionnaire for evaluating a candidate's technical leadership, problem-solving, and team collaboration."
- "Create an executive reference check guide with targeted follow-up probes for evaluating strategic decision-making and accountability."
- "Write candidate reference request guidelines explaining how candidates should select manager, peer, and direct report references."
- "Draft a reference check red flag identification matrix highlighting inconsistent tenure details, hesitant responses, or conflicting performance feedback."
- "Create a reference check summary template for presenting verified reference insights to hiring managers."
- "Draft a legal compliance reminder guide for conducting reference checks within privacy regulations."
