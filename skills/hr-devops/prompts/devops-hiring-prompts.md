# DevOps hiring prompts

Use these prompts when recruiting, screening, and evaluating DevOps engineers, Site Reliability Engineers, and infrastructure professionals.

- "Draft first-round technical screening questions to evaluate candidate experience with Kubernetes, Terraform, container orchestration, and CI/CD pipelines."
- "Create an interview scorecard section for evaluating infrastructure-as-code proficiency, security automation, and observability toolsets."
- "Write a practical take-home assignment prompt for a Senior DevOps Engineer role focused on designing automated deployment pipelines."
- "Draft a comparison guide explaining cloud containerization and deployment tools for recruiters assessing candidate experience across AWS, Azure, and GCP."
- "Create behavioral interview questions probing how an engineer led incident management and post-mortem analysis during high-severity production outages."
- "Draft a job description for a Senior DevOps Engineer clearly detailing on-call rotation responsibilities, automation goals, and infrastructure tooling."
