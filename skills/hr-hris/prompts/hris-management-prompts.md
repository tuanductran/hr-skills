# HRIS management prompts

Use these prompts when implementing, migrating, configuring, and maintaining Human Resources Information Systems.

- "Draft an HRIS evaluation requirement matrix comparing leading core HR platforms based on global payroll integration, reporting capabilities, and self-service UX."
- "Create an HRIS data migration checklist outlining data cleanup steps, field mapping validation, and security permissions checks before system cutover."
- "Write an end-user training and change enablement plan for rolling out a new employee self-service HRIS portal to managers and staff."
- "Draft a vendor RFP document for selecting a cloud HRIS vendor, including requirements for security compliance, API access, and customer support SLAs."
- "Create an HRIS access control policy guide establishing role-based permissions, audit logging, and data privacy protocols for sensitive employee records."
- "Draft an HRIS post-implementation audit template to evaluate user adoption rates, data accuracy metrics, and system configuration issues."
