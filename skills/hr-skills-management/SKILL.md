# HR skills management

Build skills-based talent strategies — from developing skills taxonomies and conducting gap analyses to designing skills-based hiring, learning pathways, and workforce capability maps that connect individual skills to organizational needs.

## Supported tasks

- Building organizational skills taxonomies and classification frameworks
- Conducting skills gap analyses at individual, team, and organizational levels
- Designing skills-based hiring and promotion processes
- Creating skills inventories and workforce capability maps
- Developing skills adjacency models for workforce reskilling
- Building skills-based learning pathway frameworks
- Designing skills assessment tools and proficiency scales
- Aligning skills strategy with workforce planning and succession planning
- Implementing skills data governance and maintenance processes
- Benchmarking organizational skills against market demand and future needs

## Key prompts

### Skills taxonomy and framework

1. "Build a skills taxonomy for [function/industry] covering technical, behavioral, and leadership skills."
2. "Create a skills classification framework with categories, subcategories, and proficiency levels."
3. "Develop a future-critical skills list for [industry] over the next 3–5 years."
4. "Design a skills ontology connecting related skills for reskilling pathway planning."
5. "Create a skills proficiency scale from foundational to expert for [skill category]."

### Skills gap analysis

1. "Conduct a skills gap analysis for [team/function] preparing for [business change]."
2. "Design a skills self-assessment survey for employees in [role family]."
3. "Build a skills heat map showing current vs. required capabilities across [organization]."
4. "Develop a skills demand forecast aligned to the 3-year business strategy."
5. "Create a skills gap priority matrix based on business criticality and gap severity."

### Skills-based talent practices

1. "Redesign the hiring process for [role] using skills-based criteria instead of credentials."
2. "Create a skills-based job description framework removing unnecessary degree requirements."
3. "Develop a skills-based promotion criteria framework for [function]."
4. "Design an internal skills marketplace connecting employees to projects by skills."
5. "Write a skills validation process for recognizing informally acquired capabilities."

### Learning and development alignment

1. "Map skills gaps to learning interventions for a [reskilling/upskilling] program."
2. "Create a skills-to-learning pathway for transitioning [role] to [emerging role]."
3. "Design a skills development plan template for individual use with managers."
4. "Build a critical skills acceleration program for [high-priority skill area]."
5. "Develop a skills progress tracking framework for measuring learning ROI."

## Tips

- Start with a focused skills taxonomy for the highest-priority roles or functions rather than attempting to map the entire organization at once.
- Use skills data to make talent decisions more objective—reduce reliance on job titles and credentials.
- Refresh the skills taxonomy annually; in-demand skills evolve faster than traditional job architecture.
- Distinguish between skills that are foundational (broad workforce need) and differentiating (competitive advantage).
- Connect skills gap data directly to L&D investment decisions to improve learning portfolio relevance.
- Involve subject matter experts in building skills taxonomies—HR alone cannot accurately define technical proficiency levels.
