# Designing accommodation request processes

Use these prompts when designing, reviewing, or improving your organization's accommodation request intake and resolution process.

- "Design a step-by-step accommodation request process from initial employee contact through implementation and follow-up, including who owns each step."
- "Draft an accommodation request intake form that collects functional limitation information without asking for a specific diagnosis."
- "Create a process map for the interactive dialogue between HR, the employee, and their manager when handling a [physical/mental health/sensory] accommodation request."
- "Write a policy statement employees can read before submitting an accommodation request that sets expectations for timeline, confidentiality, and the interactive process."
- "What documentation should HR require, and what should HR never request, when an employee submits an accommodation request for [a visible disability/a non-apparent condition/a temporary injury]?"
- "Design an accommodation request tracking system so HR can monitor open cases, response times, and outcomes without storing medical information in the wrong places."
- "Create an escalation protocol for accommodation requests that have been pending more than [14/30] business days without resolution."
