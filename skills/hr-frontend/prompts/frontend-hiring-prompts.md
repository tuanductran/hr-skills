# Frontend hiring prompts

Use these prompts when recruiting, assessing, and evaluating frontend software engineering candidates.

- "Draft first-round screening questions to evaluate candidate experience with React, TypeScript, state management, and web performance optimization."
- "Create an interview scorecard rubric for evaluating web accessibility standards, modern CSS architecture, and browser rendering optimization."
- "Write a practical take-home assignment brief asking a candidate to build a responsive component library with unit tests."
- "Draft a comparison guide of frontend JavaScript frameworks such as React, Vue, and Angular for recruiters evaluating candidate profiles."
- "Create behavioral interview questions probing how a frontend engineer collaborated with product designers and resolved cross-browser compatibility issues."
- "Draft a job description for a Senior Frontend Engineer highlighting user interface design systems, web performance metrics, and application architecture."
