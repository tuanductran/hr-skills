# Fullstack hiring prompts

Use these prompts when recruiting, assessing, and evaluating fullstack engineering candidates.

- "Draft first-round technical screening questions to evaluate candidate balance between frontend UI development and backend API architecture."
- "Create an interview scorecard rubric for evaluating fullstack architecture, database design, API integration, and frontend component design."
- "Write a practical take-home coding challenge brief asking a candidate to build a fullstack feature with React and Node.js."
- "Draft a candidate evaluation guide distinguishing fullstack engineers with frontend bias versus backend bias for hiring managers."
- "Create behavioral interview questions probing how a fullstack engineer prioritized feature trade-offs between backend security and frontend user experience."
- "Draft a job description for a Lead Fullstack Engineer emphasizing end-to-end feature ownership, technical stack alignment, and team mentorship."
