# Hiring a Gameplay Engineer for a Live-Service Title

## Context

A mid-size studio is scaling its live-service game team and needs a mid-level Gameplay Engineer experienced with Unreal Engine and live game operations.

## Step 1: Write the job description

Sample prompt: "Write a job description for a mid-level Gameplay Engineer role using Unreal."

Expected response: A job description emphasizing live-service experience (post-launch content updates, live-ops tooling) alongside core gameplay programming skills in Unreal Engine.

## Step 2: Design the technical interview

Sample prompt: "Design a technical interview assessing engine/gameplay programming skills in Unreal" and "Draft a coding exercise relevant to real-time rendering or gameplay systems programming."

Expected response: An interview combining a gameplay systems coding exercise (e.g., implementing a simple ability/cooldown system) with a discussion of the candidate's live-service operational experience.

## Step 3: Verify with references

Sample prompt: "Draft reference-check questions focused on a candidate's specific contribution to a shipped title."

Expected response: Reference questions asking a former lead to describe the candidate's specific ownership area within the shipped title and how they handled a live post-launch issue under time pressure.

## Workflow summary

The studio writes a role-specific job description, tests genuine gameplay engineering skill through a relevant coding exercise, and verifies real shipped-title contribution through targeted reference questions.
