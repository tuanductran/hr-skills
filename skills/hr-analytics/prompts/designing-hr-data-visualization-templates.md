# Designing HR data visualization templates

This AI assistant can be a powerful tool to help you with the task of designing HR data visualization templates. As an AI language model, it can generate prompts and provide suggestions to inspire your creativity and streamline your design process. With the assistant, you can easily access a vast pool of knowledge and expertise, and get instant feedback on your designs.

- "I need to create a [headcount planning dashboard] that shows our [workforce] numbers across multiple [departments], [locations], and [job levels]. Can you suggest a chart type that can handle this complexity and provide any tips on how to optimize its design for readability?"
- "I'm designing an [employee engagement survey] report that needs to display [engagement] levels by [age], [gender], [tenure], and [department]. Can you suggest some best practices for segmenting data and designing a layout that facilitates comparison and analysis?"
- "I want to create a [recruiting pipeline visualization] that tracks [applicant] data from [sourcing] to [hiring]. Can you suggest a format that can show multiple stages of the pipeline and any tips on how to make it interactive and engaging?"
- "I'm working on a [diversity scorecard] that needs to track [diversity] metrics by [race], [ethnicity], [gender], and [disability] status across our workforce. Can you suggest a format that allows for drill-down analysis and highlight any best practices for communicating sensitive data?"
- "I need to create a [benefits utilization report] that shows how our [benefits] program is being used by employees across [location], [age], and [job level]. Can you suggest a chart type that can convey this information clearly and any tips for highlighting key insights?"
